Table Scraper - Privacy Policy

Last Updated: May 15, 2025

Overview

Table Scraper is a Chrome extension designed to extract tabular data from webpages and convert it to CSV format. This privacy policy explains how we handle data when you use our extension.

Information We Collect

We collect the following information:

  • Website Content: We temporarily process the HTML content of elements you explicitly select for extraction. This data is sent to our server for processing into CSV format.

How We Use Information

The website content you select is used solely to:

  • Process and convert the selected HTML tables into structured CSV format
  • Return the processed CSV data to you within the extension

We do not store this data permanently after processing is complete.

Data Sharing

We do not sell, trade, or otherwise transfer your selected data to third parties.

Data Security

We implement reasonable security measures to protect the data you select during transmission to our servers.

Your Rights

Since we don't store your data permanently, there is no need for access, correction, or deletion requests.

Changes To This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page.
